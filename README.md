**Project Title**

A Trip To Hogwarts

**📌 Project Overview**

This project is an interactive Harry Potter–inspired experience designed to make the user feel like they are part of a magical environment. It is built as a three-stage system: a sorting hat quiz using audio and buttons, a wand selection process using LED feedback, and a final interaction stage where the user controls objects using wand gestures.

The system uses ESP32 microcontrollers, motion sensors, LEDs, motors, and audio modules to create real-time responses to user actions. The focus of the project is on interaction and feedback, where every input from the user leads to a visible or physical output, recreating the feeling of magic in a simple, tangible way.
